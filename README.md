# Networkee MVP v1

Ein erster klickbarer Web-/PWA-Prototyp für Networkee als persönliches CRM.

## Funktionen
- Dashboard mit Kontakt-, Interaktions- und Follow-up-Übersicht
- Kontaktverwaltung mit Rolle, Firma, Tags, Ziel und Merkhilfe
- Interaktionserfassung mit Notiz, nächstem Schritt und Follow-up-Datum
- Relationship Insights auf Basis vorhandener Notizen
- Lokale Speicherung im Browser via localStorage
- JSON-Export
- PWA-Grundstruktur mit Manifest und Service Worker

## Start lokal
Datei `index.html` im Browser öffnen.

## Wichtige Einschränkung
Dies ist bewusst ein MVP ohne Backend, Login, Datenbank, KI-API, Kalenderintegration oder Payment. Diese Punkte gehören in spätere Iterationen.
